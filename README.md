##Sobre O crud

- As questões podem ser vizualizadas atraves do menu, cliando em questões 

-Configurações do arquivo `.env`:

    DB_CONNECTION=pgsql
    DB_HOST=127.0.0.1
    DB_PORT=5432
    DB_DATABASE=mauricio
    DB_USERNAME=postgres
    DB_PASSWORD=senha


- Rodar os comandos 
- migrate 
    `php artisan migrate`

- Seed para popular o banco
    `php artisan db:seed`
